## 2018/05/18

- 深入学习`_debounce`(弹簧合并函数) 与 `_throttle`(函数节流)

- 阅读lodash函数库

``` js
// 执行1次(最后一次点击1000ms后)
btnDom.addEventListener('click', debounce(clickBtn, 1000)); 
// 执行3次(点击立即执行一次、1000ms后执行一次，2000ms后执行一次)
btnDom.addEventListener('click', throttle(clickBtn, 1000)); 
```
